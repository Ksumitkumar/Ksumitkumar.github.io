About

From native Android to cross-platform Flutter

I started out writing native Android in Kotlin and Java — API integrations, push notifications, geolocation, and camera-based features — then moved into Flutter, where I've spent the last few years building cross-platform apps end-to-end: UI, state management, local storage, and backend integration.

Most of my work has shipped for e-commerce, real estate, and service marketplaces, with a recurring focus on payments (Razorpay, Stripe, Instamojo), Firebase OTP authentication, and offline-first local databases like SQLite.
